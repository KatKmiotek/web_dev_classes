View console output

This example prints messages to the browser console when the page loads. To see them, follow the steps below:

  1. In VS Code, right-click index.html and choose Open in Default Browser (or use the extension/command that opens the file in your default browser).
  2. When the page opens in the browser, open the Developer Tools (usually F12, Cmd+Option+I on macOS, or right-click → Inspect).
  3. Switch to the Console tab — you'll see the messages printed by script.js (they run automatically on page load).

If you don't see any messages, reload the page with the Console open.

Tip: script.js logs a few messages when the page loads so you can test the Console immediately.
