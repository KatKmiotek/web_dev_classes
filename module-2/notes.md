### 3. Essential HTML Elements

**Practice**:
1. Add text in index.html file
2. Open index.html file in the default browser
3. Right-click on file > Open in default browser


### Text elements
  - Headings (h1-h6) 
```html
<h1>My heading</h1>
```
  - Paragraphs (p)
```html
<p>a little text here</p>
```
  - Line breaks (br) and horizontal rules (hr)
- Text formatting
  - Bold (strong) and italic (em)
```html
<strong>Important text here</strong>
<em>Italic text</em>
```
- Lists
  - Unordered lists (ul)
  - Ordered lists (ol)
  - List items (li)

```html
<p>Shopping list</p>
<ul>
  <li>fruits </li>
  <li>water</li>
</ul>  


<p>Shopping list</p>
<ol>
  <li>fruits </li>
  <li>water</li>
</ol> 
```
